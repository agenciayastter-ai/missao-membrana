# Missão Membrana

Projeto React pronto para publicação na Vercel.

## Como rodar no computador

1. Instale o Node.js.
2. Abra a pasta do projeto no terminal.
3. Execute:

```bash
npm install
npm run dev
```

## Como publicar na Vercel

1. Acesse https://vercel.com/
2. Crie sua conta.
3. Clique em **Add New Project**.
4. Envie esta pasta do projeto ou conecte um repositório Git.
5. A Vercel detecta Vite/React automaticamente.
6. Clique em **Deploy**.

## Build

```bash
npm run build
```
